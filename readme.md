
# Triangles

## Описание

Проект представляет из себя программу для определения пересечения треугольников в 3D. На вход подаётся количество треугольников, а дальше координаты треугольников в последовательности `x1 y1 z1 x2 y2 z2 x3 y3 z3`, пример: 
```
2

-86.9843 -41.704 240.975
-123.805 -37.7712 242.059
-117.274 -65.7075 284.955

53.871 104.777 -149.02
71.236 128.091 -194.093
55.8972 133.49 -165.948
```
> Примеры наборов данных можно найти в папке `./end2end/tests`

В качестве выходных данных выдаются индексы пересекающихся треугольников.

## Сборка

Зависимости проекта (c++20):

```bash
sudo apt install -y \
    cmake \
    build-essential \
    libgtest-dev \
```

Сборка проекта осуществляется следующими командами: 

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
```

Запуск:
```bash
./build/triangles
```

Запуск на примерах из папки `./end2end`:
```bash
./build/triangles < ./end2end/tests/test1.txt
```

Запуск тестов:
```bash
ctest --test-dir build/tests --output-on-failure
```
## Отладка

Чтобы скомпилировать программы в режиме отладки, следует скомпилировать в режиме `Debug`

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Debug
cmake --build build
```
